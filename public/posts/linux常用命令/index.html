linux常用命令命令

制作linux usb启动盘

格式化u盘

mkfs.vfat -F 32 -n USB /dev/sdb1

安装syslinux,mtools

apt-get install syslinux mtools

安装linux引导

/usr/bin/syslinux /dev/sdb1

写入mbr

cat /usr/lib/syslinux/mbr/mbr.bin > /dev/sdb

配置重命名

mv ioslinux.cfg syslinux.cfg
